# ublock-custom-filters

My **uBlock Origin** filters for readability, privacy, and system-syncing.

* **`hackernews`**: OS-aware Dark Mode, **16px** scaling
* **`google-auth-popup`**: Global network block for "Sign in with Google" prompts.

## Installation

1. Open **uBlock Origin Dashboard** > **My filters**.
2. Paste file contents.
3. Click **Apply changes**.
